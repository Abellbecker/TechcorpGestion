Características principales
Gestión de Perfiles: Registro de nombre, apellido, salario y otros datos personales.

Jerarquía de Clases: Implementación de distintos tipos de empleados (Desarrolladores, Gerentes, Administrativos, etc.).

Cálculos Automáticos: Lógica para el cálculo de salarios según el rol o bonificaciones específicas.

Persistencia de datos: (Opcional, si lo tienes) Carga y lectura mediante consola o archivos.

Conceptos de Programación Aplicados
Para este proyecto me enfoqué en aplicar conceptos sólidos de ingeniería de software:

Herencia: Clase base Empleado y subclases específicas.

Polimorfismo: Métodos sobrescritos para comportamientos específicos según el rol.

Encapsulamiento: Protección de datos sensibles mediante modificadores de acceso.

Requisitos
Java JDK 11 o superior.

Un IDE (IntelliJ IDEA, Eclipse, VS Code).
